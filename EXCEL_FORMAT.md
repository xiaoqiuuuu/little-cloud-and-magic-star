# Excel 题目导入导出格式说明

本文档说明题目管理系统中 Excel 批量导入导出功能的使用方法和格式要求。

## 📥 导入功能

### 使用方式

1. 点击"**下载导入模板**"按钮，获取标准格式的 Excel 模板
2. 在模板中填写题目数据
3. 点击"**批量导入题目**"按钮，选择填写好的 Excel 文件
4. 确认导入后，系统会自动处理并显示导入结果

### Excel 表格格式

Excel 文件必须包含以下列（列名必须完全一致）：

| 列名 | 是否必填 | 说明 | 示例 |
|------|---------|------|------|
| ID | 否 | 题目唯一标识符。<br>- 留空：创建新题目<br>- 填写已有ID：更新现有题目 | `q_001` |
| 题目 | **是** | 题目内容 | `小云是哪一年出生的？` |
| 答案 | **是** | 题目答案（不区分大小写） | `2000` |
| 资源链接（多个用\|分隔） | 否 | 相关资源的URL链接<br>多个资源用竖线 `\|` 分隔 | `https://example.com/image.jpg\|https://example.com/video.mp4` |
| 标签（concert/vlog/common） | 否 | 题目分类标签<br>可选值：`concert`、`vlog`、`common`<br>默认：`common` | `concert` |
| 出题人（多个用\|分隔） | 否 | 出题人姓名<br>多个出题人用竖线 `\|` 分隔 | `制作人A\|制作人B` |
| 随机点击数 | 否 | 统计数据（导入时可忽略） | `15` |
| 隐藏点击数 | 否 | 统计数据（导入时可忽略） | `8` |

### 数据填写规范

#### 1. 必填字段
- **题目**和**答案**为必填项，不能为空
- 如果有空白，导入时会提示错误行号

#### 2. ID 字段说明
- **留空**：系统会自动创建新题目并生成新的ID
- **填写已存在的ID**：更新该ID对应的题目
- **填写不存在的ID**：导入会失败并提示错误

#### 3. 资源链接格式
- 支持图片、视频、音频等多种资源类型
- 多个链接用英文竖线 `|` 分隔
- 示例：
  ```
  https://example.com/image1.jpg|https://example.com/image2.jpg|https://example.com/video.mp4
  ```

#### 4. 标签值
- 只能是以下三个值之一：`concert`、`vlog`、`common`
- 不区分大小写
- 填写其他值会自动转为 `common`

#### 5. 出题人
- 可以填写多个出题人，用竖线 `|` 分隔
- **如果出题人不存在，系统会自动创建并添加到制作人列表**
- 建议使用制作人管理中已存在的制作人名称
- 示例：
  ```
  张三|李四|王五
  ```

### 导入示例

#### 示例1：创建新题目
```
ID: (留空)
题目: 小云参加过哪个音乐节？
答案: 草莓音乐节
资源链接（多个用|分隔）: https://example.com/concert.jpg
标签（concert/vlog/common）: concert
出题人（多个用|分隔）: 制作人A
随机点击数: (留空)
隐藏点击数: (留空)
```

#### 示例2：更新已有题目
```
ID: q_123
题目: 小云参加过哪个音乐节？（修改后）
答案: 草莓音乐节
资源链接（多个用|分隔）: https://example.com/concert.jpg|https://example.com/concert2.jpg
标签（concert/vlog/common）: concert
出题人（多个用|分隔）: 制作人A|制作人B
随机点击数: 10
隐藏点击数: 5
```

### 导入结果

导入完成后会显示：
- ✅ 成功导入的题目数量
- ❌ 失败的题目数量和详细错误信息
- 失败原因可能包括：
  - 必填字段为空
  - ID不存在（更新时）
  - **题目重复**（题目和答案完全相同的题目已存在）
  - 数据格式不正确
  - 网络或服务器错误

## 📤 导出功能

### 使用方式

1. 点击"**导出全部题目**"按钮
2. 系统会自动下载包含所有题目的 Excel 文件
3. 文件名格式：`题目导出_YYYY-MM-DD.xlsx`

### 导出内容

导出的 Excel 文件包含当前系统中的所有题目，包括：
- 完整的题目信息（ID、题目、答案等）
- 资源链接（多个资源用 `|` 分隔）
- 标签和出题人信息
- 统计数据（随机点击数、隐藏点击数）

### 用途

导出的文件可以用于：
1. **数据备份**：定期导出题目数据作为备份
2. **批量修改**：在 Excel 中批量编辑题目后重新导入
3. **数据迁移**：导出后在其他系统中导入
4. **数据分析**：使用 Excel 分析题目数据和统计信息

## 💡 使用技巧

### 1. 批量创建题目
- 下载模板后，在 Excel 中复制模板行
- 批量填写多道题目的信息
- ID 列留空，一次性导入创建多道题目
- **注意**：系统会自动检查重复题目（题目和答案完全相同），重复的题目会被跳过

### 2. 批量更新题目
- 先导出所有题目
- 在导出的文件中修改需要更新的题目
- 重新导入，系统会根据 ID 自动更新

### 3. 混合操作
- 同一个 Excel 文件中可以同时包含新建和更新操作
- 有 ID 的行会更新，无 ID 的行会创建

### 4. 数据校验
- 导入前在 Excel 中检查数据完整性
- 确保必填字段都已填写
- 检查标签值是否正确（concert/vlog/common）

### 5. 错误处理
- 如果导入失败，查看错误详情
- 根据提示修改 Excel 文件中的数据
- 重新导入直到成功

## ⚠️ 注意事项

1. **文件格式**：只支持 `.xlsx` 和 `.xls` 格式的 Excel 文件
2. **列名匹配**：Excel 中的列名必须与模板完全一致（包括括号和标点符号）
3. **数据验证**：导入前建议先导出查看当前数据格式
4. **备份数据**：批量操作前建议先导出当前数据作为备份
5. **分批导入**：如果题目数量特别多，建议分批导入以避免超时
6. **网络稳定**：导入大量数据时需要稳定的网络连接
7. **特殊字符**：避免在数据中使用 `|` 符号（该符号用作分隔符）

## 🔧 常见问题

### Q1: 导入时提示"第X行的题目或答案为空"
**A:** 检查 Excel 文件第X行，确保"题目"和"答案"两列都有内容。

### Q2: 导入后部分题目失败
**A:** 查看错误详情，通常是因为 ID 不存在或数据格式不正确。修改后重新导入失败的题目即可。

### Q3: 如何批量删除题目？
**A:** 目前不支持通过 Excel 批量删除。建议在系统界面中逐个删除，或者导出后筛选需要保留的题目，清空数据库后重新导入。

### Q4: 资源链接过长怎么办？
**A:** 可以在 Excel 中调整列宽，或者使用短链接服务缩短URL。

### Q5: 出题人不存在怎么办？
**A:** 不用担心！系统会自动检查出题人是否存在，如果不存在会自动创建并添加到制作人列表中。导入完成后，可以在"制作人管理"中查看和编辑这些制作人的信息。

### Q6: 批量导入会影响现有数据吗？
**A:** 
- 如果 Excel 中 ID 列为空，会创建新题目，不影响现有数据
- 如果填写了已存在的 ID，会更新该题目，其他题目不受影响
- 建议导入前先导出备份
